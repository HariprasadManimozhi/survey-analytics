Python - 
  * Dataframe - pandas
  * Factor analysis - prince
  * Visualization - seaborn

2. Load Data
3. Explore Data
4. Data cleaning

5. Feature Engineering

    * Cross tabulation
6. Exploratory Data Analysis

    * Cross analysis
7. Derive Insights
